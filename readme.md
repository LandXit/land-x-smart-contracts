# LandX-it Smart Contracts

Nodejs: 14.17.0

##

Compile

```shell
npx truffle compile
```

### Ganache

Deploy
```shell
npx truffle migrate --network ganache
```

Connect
```shell
npx truffle console --network ganache
```

### Ropsten

Deploy
```shell
npx truffle migrate --network ropsten
```

Connect
```shell
npx truffle console --network ropsten
```
